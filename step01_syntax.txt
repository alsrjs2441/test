3. 실생활에서 표현 가능한 데이터 포맷들
	-예시
		숫자 정수 10데이터를 age라는 이름의 변수명으로 저장 및 관리
		"		22 데이터는 age2라는 """"		"
		파이썬은 타입 선언을 안함 <-> 자바는 타입 선언해야함
	1. 숫자
		실수
			float(32bit)/double(64bit)
		정수
			byte(8bit)/short(16bit)/int(32bit)/long(64bit)
	2. 문자
		문자 char
		문자열 string
	3. 논리
		true/false boolean
		
----- syntax ------

* 하나의 자바 소스에서 문법 숙지
1. 변수
	1-1. 변수 선언 문법(구현)
		타입 변수명;		선언
		타입 변수명 = 값;	선언 및 초기화
		
	1-2. 변수 호출 문법
		변수명 = 값; 호출해서 값 대입
		System.out.println(변수); 변수값을 출력
		
2. 메소드
	2-1. 구현 문법
		호출한 곳으로 되돌려 줄 타입 기능이름(입력 데이터 받을 변수, ){
			기능 구현
			return 값;
		}
	2-2. 호출 문법
		객체지향 언어인 자바의 메소드 호출은 무법적으로 객체 라는 거슬 만든 후에 호출 가능
		객체는
		- 메모리에 클래스들이 보유한 변수나 메소드를 사용 할 수 있도록 주제
		- 생성 문법
			new 클래스명()

	*서로 다른 클래스들 간에 기능 및 데이터 호출해서 활용

	- 참조 타입 활용 = 객체 생성 및 활용
	
4.생성자
	1. 용도 : 객체를 생성할 수 있게 하는 구성요소

			참고 - 객체를 만들고 사용하는 의미는 데이터를 명확하게 구분해서 관리 및 활용
	2. 호출하는 시점이 가장 먼저 호출되는 구성 요소
		
5. 데이터 타입이 사용 가능한 문법 위치
	1. 변수 타입 - 타입 변수명
	2. 메소드의 반환 타입 - 리턴타입 메소드명([..]){[..]}
	3. 메소드의 argument or parameter - 리턴 타입 메소드명 (변수 타입 변수명 [,..]){}
	4. 생성자의 argument or parameter - class명(변수타입 변수명[,..]){}

6. 타입 종류
	기본 타입(8가지)/ 참조 타입(모든 class를 기반으로 활용)	
------
1. 데이터 타입
	1. 기본 타입(8가지)
		숫자 (실수, 정수),문자,논리
		
	2. 참조 타입(객체, reference)
		- 기본 타입을 제외한 모든 타입
		- 사용자(개발자)들이 만드는 모든 클래스들 자체가 타입
		- API에서 제공하는 클래스들
		- 사용을 위한 전제조건 : 문법적으로 메모리에 사용 가능하게 생성
			클래스명 변수 = new 클래스명();
			Step02Method s = new Step02Method();
			
			Step02Method 타입의 s라는 변수 공간 생성
			new Step02Method() 객체 생성
			= 생성된 객체를 s변수에 대입
			; 해당 로직을 마무리			
		...
